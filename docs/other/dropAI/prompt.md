AI Dropshipping Product Finder – SaaS MVP UI Design

⸻

🧠 CONTEXT

Design a complete SaaS dashboard UI for an AI-powered Dropshipping Product Finder.

This tool helps ecommerce founders find winning dropshipping products using AI trend analysis, profit estimation, and supplier matching.

The UI must be modern, minimal, conversion-focused, similar to tools like Tradelle, Dropship.io, Zendrop, ZIK Analytics.

Design for desktop first.

⸻

🎯 TARGET USERS
• Shopify store owners
• Beginner dropshippers
• Ecommerce agencies
• Product researchers

Skill level: non-technical users.

Goal: quickly find winning products.

• Primary: #4F46E5 (Indigo)
• Secondary: #06B6D4 (Cyan)
• Background: #0F172A (Dark)
• Card Background: #1E293B
• Text Primary: #F8FAFC
• Accent: #22C55E (Profit indicators)

Typography:
• Headings: Inter Bold
• Body: Inter Regular
• Numbers: Mono font for metrics

Spacing: generous padding, rounded cards, soft shadows.

Design should feel like:
Stripe Dashboard + Notion + Linear.
